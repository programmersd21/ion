# Security Policy for @pro-grammer-SD Projects

Maintained by Soumalya Das  
Email: geniussantu1983@gmail.com

## Supported Versions
We actively maintain the latest stable releases of all projects.  
Security updates are provided for the most recent release and the previous major release.

## Reporting a Vulnerability
Please do not open public issues for security vulnerabilities.  
Report issues via email: **geniussantu1983@gmail.com**

Include the following in your report:
- Description of the issue
- Steps to reproduce
- Potential impact
- Any proposed mitigation

Expected response time: 48–72 hours.

## Submitting Security Patches
- Fork the repository
- Implement the fix in a branch
- Submit a pull request referencing the security report
- Ensure all tests pass
- Maintainers will review and merge with priority

We appreciate responsible disclosure and will credit contributors appropriately.
